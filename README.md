# DSA-Java

Daily Data Structures and Algorithms practice in Java.

## Goals

- Build strong problem-solving skills with consistent practice.
- Cover core DSA patterns topic by topic.
- Keep each solution simple, readable, and interview-focused.

## Project Structure

```text
src/
  day01/
    LargestElement.java
```

Each `dayXX` folder contains practice problems and solutions for that day.

## How to Run

From the project root:

```bash
javac src/day01/LargestElement.java
java -cp src day01.LargestElement
```
